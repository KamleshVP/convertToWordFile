# convertToWordFile

Program for converting numeric value to word.

## Installation

Use Eclipse or notepad++ with jdk version 1.8v.

```
install eclipse
```

## Usage

```
User Input - 123456.78
Expected Output - Rs. One Lakh Twenty Three Thousand Four Hundred And Fifty Six 78/100 ONLY
```
